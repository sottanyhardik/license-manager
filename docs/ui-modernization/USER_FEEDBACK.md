## How to add feedback

When the user reviews a READY_FOR_REVIEW route and requests changes:

1. Add an entry below under **Outstanding**
2. Return the route to `IN_PROGRESS` in ROUTES.md
3. When addressed, mark ✅ and move to **Resolved**

```
### [Route] `/route-path` — [Date]
**Requested by:** User
**Description:** What they asked for
**Priority:** P0/P1/P2
**Status:** Outstanding / ✅ Resolved in Session N
```
